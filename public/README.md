# Landing Page Files

## Production File

**landing.html** - Production-ready landing page with:
- ✅ Real email collection (Web3Forms integration)
- ✅ Error handling & loading states
- ✅ SEO optimized meta tags
- ✅ Social sharing (Open Graph + Twitter Cards)
- ✅ Analytics ready (Google Analytics + Plausible)
- ✅ Spam protection (honeypot field)
- ✅ Responsive design
- ✅ Accessibility compliant

## Quick Start

1. **Get Web3Forms key**: https://web3forms.com/
2. **Edit landing.html**: Add your access key (line 1017)
3. **Deploy**: `npx vercel --prod`

## Live URL

After deployment, access at:
```
https://your-project.vercel.app/landing.html
```
